# ⚽ Mundial 2026 - Gestor de Resultados

Una aplicación web interactiva para gestionar resultados de partidos, tabla de posiciones, gráficos estadísticos y cuadro de cruces del Mundial 2026.

## 🎯 Características

- **Registrar Resultados**: Carga partidos con goles de ambos equipos
- **Tabla de Posiciones**: Visualiza la clasificación de todos los grupos automáticamente
- **Estadísticas y Gráficos**: 
  - Gráficos de puntos por grupo
  - Comparativa goles anotados vs recibidos
  - Top 10 equipos y goleadores
  - Diferencia de goles
  - Estadísticas detalladas por grupo
- **Cuadro de Cruces**: 
  - Dieciseisavos de Final
  - Octavos, Cuartos, Semifinales
  - Final y Tercer Lugar
  - Se actualiza automáticamente con los resultados

## 🚀 Uso

1. **Descargar o clonar el repositorio**
   ```bash
   git clone https://github.com/tu-usuario/mundial2026.git
   cd mundial2026
   ```

2. **Abrir en el navegador**
   - Haz doble clic en `index.html`
   - O arrástralo a tu navegador

3. **Usar la aplicación**
   - Selecciona grupo, equipos y goles
   - Haz clic en "Registrar Partido"
   - Visualiza posiciones, gráficos y cruces
   - Los datos se guardan localmente en tu navegador

## 📱 Tecnologías

- **Frontend**: HTML5, CSS3, JavaScript vanilla
- **Gráficos**: Chart.js
- **Almacenamiento**: localStorage (sin servidor necesario)
